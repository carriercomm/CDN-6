CDN
===

authors: Andrew Barba and Gary Soeller

DNS Server
=========

```./dnsserver -p <port> -n <name>```

HTTP Server
===========

```./httpserver -p <port> -o <origin>```

Everything else
===============

```./[deploy|run|stop]CDN -p \<port\> -o \<origin\> -n \<name\> -u \<username\> -i \<keyfile\>```
